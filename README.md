# TikTok-SnakeBot

## Technologies used
Python - Selenium, Moviepy  
Java - Processing for rendering/render server

## General Functionality:  
1. Render the State
2. Upload the State
3. Wait 10 minutes
4. Retrieve [likes, comments, shares]
5. Move Snake accordingly

Like - Move forward | Comment - Move left | Share - Move right
